# Versions

Built `.gba` ROMs of the SAG birthday project, one folder per release.

| Version | Notes |
|---|---|
| v0.1.0-baseline | Vanilla pokeemerald-expansion + the Shane / Justin / Hudson Yards renames. **No SAG content yet** — this just proves the toolchain/CI works. The full game (teams, maps, story, systems from `SAG_BIRTHDAY_PLAN.md`) is implemented in later versions. |

ROMs are built in the cloud by the **Release ROM** GitHub Action (no local toolchain needed), then committed here.
